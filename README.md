# spam-call-detector
Application which identifies spam calls

## Setup

##### Create a virtual environment

On linux/mac

```bash
python3 -m venv env
```

On windows

```
python -m venv env
```

##### Activate the virtual environment

On linux/mac

```bash
source env/bin/activate
```

On windows

```bash
.\env\Scripts\Activate.ps1
```

##### Install dependencies

```bash
pip3 install -r requirements.txt
```

##### To run the app

```bash
python app/main.py
```
